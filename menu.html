How to play The Royal Vault

What is it?

The Royal Vault is a memory game

There are 3 different levels: Easy, Medium, Hard

The number of cards displayed increases as the difficulty increases

How to Play

There are flipped cards displayed on each level

To flip a card you click on it

You flip 2 cards and if they are a pair they remain flipped

If they are not a pair they flip back over

The score increases for every pair that is flipped

There is a timer every game

Every correct pair increases the timer by 3 seconds

Aim of the Game

The goal of the game is to get the lowest score

Depending on the Level the score for a Win varies

If the timer runs out, the game ends and you Lose

Easy Level

Win: 4-7 | Average: 8-10 | Lose: 11+ | Timer: 30 seconds

Medium Level

Win: 8-14 | Average: 15-22 | Lose: 23+ | Timer: 35 seconds

Hard Level

Win: 12-22 | Average: 23-32 | Lose: 32+ | Timer: 50 seconds

Reasoning behind the Game

Nowadays our attention span has shortened

And our memory has become worse over the years

To help improve our memory I have developed a memory game to try to help with this problem
